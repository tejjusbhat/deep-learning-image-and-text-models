README

All final written reports for this coursework are contained within the report notebooks located in the Part 1 and Part 2 folders respectively.

The model training notebooks and checkpoints are also included inside their corresponding Part 1 and Part 2 directories. However, these training notebooks are not recommended to be executed. They were used during development and may not run correctly due to path changes, environment differences, or experimental code structure.

The report notebooks contain:

* The completed analysis and written report
* Model evaluation results
* Visualisations
* Demonstration code for loading checkpoints and generating outputs

These report notebooks represent the final, clean submission intended for assessment.
